# PowerBI Data Science Jobs Dashboard

![Dashboard Page 1](/images/Page1.png)

## Introduction

This dashboard was created for **Job Seekers and Job Transitioners** to solve the problem of data job market ambiguity. Using a **real-world dataset of 2024 data science job postings**, it provides a single, easy-to-use interface to explore market trends and compensations.

## Skills Showcased

- **Data Transformation (ETL) with PowerQuery:** cleaned, shaped and prepared the raw data for analysis (data type formatting, missing value handling, creating auxiliary tables).

- **Implicit Measures:** formulated measures to derive key insights and KPIs like median salary and job count.

- **Core Charts:** various chart types have been implemented to maximize readability, including **Column, Bar, Line and Area Charts**.

- **Geospatial Analysis:** used **Map Charts** for global distribution.

- **KPI Indicators & Tables:** used **Cards and Tables** to display key metrics.

- **Interactive reporting:** 
    - **Slicers** to dynamically filter the report by Job Title.
    - **Buttons and Bookmarks** for seamless navigation.
    - **Drill-Through** to navigate from a high-level summary to contextual, detailed view.
